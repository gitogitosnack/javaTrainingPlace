#マークダウン記法の書き方
## 
###
## 📝 マークダウン 記述法チートシート

| 目的 | 記述法 | 表示例 |
| :--- | :--- | :--- |
| **見出し (Heading)** | `# 見出し 1` <br> `## 見出し 2` <br> `### 見出し 3` | # 見出し 1<br>## 見出し 2<br>### 見出し 3 |
| **太字 (Bold)** | `**太字のテキスト**` | **太字のテキスト** |
| **斜体 (Italic)** | `*斜体のテキスト*` または `_斜体のテキスト_` | *斜体のテキスト* |
| **太字と斜体** | `***太字で斜体***` | ***太字で斜体*** |
| **取り消し線** | `~~取り消し線~~` | ~~取り消し線~~ |
| **箇条書き (Unordered List)** | `* アイテム 1` <br> `* アイテム 2` <br> または <br> `- アイテム 1` <br> `- アイテム 2` | * アイテム 1<br>* アイテム 2 |
| **番号付きリスト (Ordered List)** | `1. アイテム A` <br> `2. アイテム B` | 1. アイテム A<br>2. アイテム B |
| **リンク (Link)** | `[リンクのテキスト](URL)` | [Google](https://www.google.com) |
| **画像 (Image)** | `![代替テキスト](画像のURL)` | ![猫の画像](cat.png) |
| **インラインコード (Inline Code)** | `` `インラインコード` `` | `git init` |
| **コードブロック (Code Block)** | \`\`\`言語名 (例: java)` <br> `... コード ...` <br> \`\`\` | ```java<br>public static void main() {}<br>``` |
| **引用 (Blockquote)** | `> 引用文` | > これは引用文です。 |
| **水平線 (Horizontal Rule)** | `---` または `***` または `___` | (画面いっぱいの線が表示されます) |
| **改行** | 行末に**半角スペースを2つ**入れる <br> （または空行を挟む） | テキスト1  <br>テキスト2 |
